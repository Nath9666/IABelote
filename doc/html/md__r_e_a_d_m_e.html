IA Belote

Description

Ce projet à pour but de faire apprendre à une IA à jouer à la belote. Pour cela, nous avons utilisé un algorithme de renforcement, le Q-learning. Il pemert également d'explorer la programtions orientée objet mais aussi le fait de develloper une IA basée sur le Q-learning.

A noter : Ce projet est un projet personnel et n'a pas pour but d'être utilisé en production.

Il existe plusieurs versions de la belote, nous avons choisi de nous baser sur la belote avec une variantes concernant la vache folle.

Dans cette variantes, si personne n'as pris durant le deuxième tour de prises alors c'est le conneur qui prends. Cela apporte une difficulté supplémentaire pour l'IA.

Installation

Pour installer le projet, il suffit de cloner le dépôt git et d'installer les dépendances.
